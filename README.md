# Liquid Glass Messenger v4 (Discord-like) — E2EE, Admin, 10GB uploads, Animations

## Новое
- 🔐 Клиентское шифрование (E2EE) на канал: общий ключ хранится только локально (LocalStorage). Сервер видит шифротекст.
- 🗂️ Админ-панель: пользователи (delete), файлы (список/открыть/delete).
- 📦 Загрузка файлов до 10 ГБ: чанки по 10 МБ, сшивка на сервере, стримовый download.
- 🎬 Анимации: сплэш при запуске, плавные анимации рейла, кнопок, хедера.

## Запуск
Server:
```bash
cd server
cp .env.example .env   # ORIGIN=http://localhost:5173
npm i
npm start
```

Client:
```bash
cd client
npm i
npm run electron-dev
```

## E2EE
- В списке каналов появится блок **Set/Generate key / Show key**.
- Сгенерируй/задай Base64 ключ — **поделись им вне чата**. Сообщения будут шифроваться/дешифроваться на клиенте.
- Сервер хранит шифротекст (`enc:<nonce>:<box>`). Без ключа на клиенте сообщение читается как `[Encrypted]`.

⚠️ Это базовая E2EE (общий ключ на канал). Для полноценной модельки с DH и обменом ключами между участниками нужно будет расширить протокол.
